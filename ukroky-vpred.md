Úkroky vpřed
============


Mřít
----

ve světle  
knot zesiná  
nám cizí  
tupost jež  
eon / dosyta  
snad duše  
nebo srdce  
*plachtí!*  
v ságu bdění  
přimknuta  
zem skonem

hlubší, a rozryta


Hodiny
------

nocí hlasy  
vrásky / zralost / krajiny  
doufat víc  
lad hlušiny  
důvtipem / Adama  
*Evy* jmeniny  
míza předků  
za dcery, za syny  
vydrží-li / rodiny  
ve větvoví  
nadešlé světa — hodiny


Fáma
----

zpatříš ji  
vzdálenost ruky  
společné *my*  
překonat veletoky  
řekl jsi nic  
je pryč

„Kdo zvojtil osud?“  
„Nesmělé noty.“

kam běží  
něžné roky  
lásky rým  
přání sloky

oči

milé dráma  
vším tělem  
*my* / dva / káva

duje  
hezký sen  
a hezčí fáma

můžu / zmůžu? / nadšení / nic  
mamonu / ideje / co / mně je  
*jist / vahou / spící / Prahou*

dluh snů / sacharin / medojedu

silhouetta / č. dva
